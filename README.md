# 🃏 POKERUZ Online Project

Ushbu loyiha real vaqt rejimida ishlovchi poker o'yinining onlayn talqinidir.

## 🚀 Loyihaning maqsadi
- Ko'p o'yinchili (multiplayer) tizimni yaratish.
- Texas Hold'em va mahalliy karta o'yinlari qoidalarini joriy etish.
- Mobil va kompyuter qurilmalari uchun to'liq moslashuvchanlik.

## 🛠 Texnik tarkib
- **Engine:** Godot / Unity / Node.js
- **Network:** WebSockets (Socket.io)
- **UI:** Custom Card Assets & Responsive Design

## 📂 Loyiha tuzilmasi
- `client/` - Foydalanuvchi interfeysi va grafikalar.
- `server/` - O'yin mantiqi va server qismi.
- `docs/` - Qo'llanmalar va o'yin qoidalari.
